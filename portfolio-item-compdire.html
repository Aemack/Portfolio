CompanyDirectory

Check it out

The application allows users to interact with the company database. Searching, creating, editing and deleting entries using a simple user interface. The user is able to look at a directory of all employees, grouped under locations, departments or with no grouping. They are also able to search for any combination of parameters.

I made this using jQuery and Bootstrap, along with custom CSS animations, to design the user interface and AJAX requests to interact with the PHP which would make changes to the mySQL database.
