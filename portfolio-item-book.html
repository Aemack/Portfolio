Writing Prompter

This is a game where users compete to get the longest streak of matching book covers to their genres. The user clicks the genre they think matches, if correct they get a point, if wrong the score is checked against the highscore and the user is offered to start again.

This was made using Node.js, Express.js and Puppeteer.js for the back-end, collecting book covers from a publishing website under each genre. I implemented this as a full-stack app as I wanted to provide the ability to save highscores. For the front-end I used fetch requests to get and post data.
