MERN Templater

Site - Login

This is a website template I set up so that whoever would be updating it would not need any coding to update information or add blog posts. By logging in the user is able to make and save changes to the site.

The front end site is made using React and makes requests for the content using axios. The express server then gets the content from MongoDB and updates the state.

To login and make changes the user must gain an access key from the server by providing a correct username and password which must be provided whith every post request.
