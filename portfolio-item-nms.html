No Man's SkyCrafting Guide

See it here

This web-app allows users to find recipes for the game No Mans Sky. The user selects one of the products from the menu and from there it will show some information and buttons to the ingredients which, when clicked, will show the ingredients for that as well.

This app is based around a switch statement that will return the ingredients and information in an array, which are then seperated and appended to the output. The smooth appearance is down to adding and removing css classes causing the blocks to fade-in.
