Writing Prompter

See it here

This app is made up of multiple pages which provide various forms of inspiration to help get through writers block. These include a Writing-Prompter, Monster-Maker, Character Creator and Quote Generator

The Writing-Prompter works by using the Reddit API to access the top posts in th subreddit r/writingprompts and takes the title of each post.

The Character-Creator uses an API of names and origins to come up with the characters name. The age is randomised betweeen 10 and 95 and the picture is taken from a stock photo API with search terms depending on the age that was landed on.

The monster Maker also makes use of Reddit's API and the subreddits r/ImaginaryBehemoths and r/ImaginaryLeviathans for the images and their titles.
