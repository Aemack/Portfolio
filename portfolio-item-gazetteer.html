Gazetteer

Check it out!

This app allows users to see information on a selected country as well as its location on a map and points of interest, which can be toggled from a menu.

I made this using Bootstrap for the interface and jQuery to update the views. To generate the map I used Leaflet along with marker cluster to generate the points of interest. The data is taken from various REST APIs using PHP and stored in a mySQL database.
